# Camille
The Camille Editor for Rodin


## Building Camille
Maven 3 is required to build the project: 
  <pre>
  cd org.eventb.texteditor.parent
  mvn clean verify 
  </pre>  
  
This will produce an updatesite in org.eventb.texteditor.repository/target

## Contributing/Bugs
